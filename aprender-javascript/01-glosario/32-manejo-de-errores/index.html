Control de flujo y Manejo de errores

El control de flujo en JavaScript define el orden en que se ejecutan las instrucciones. Esto puede depender de estructuras como condicionales, bucles y excepciones.

Ejemplo
Línea 01| let nombre = "Pompompurin";
Línea 02| console.log(nombre);

En este caso, no hay ninguna estructura de control que altere el orden de ejecución. Sin embargo, en el siguiente ejemplo, el flujo de ejecución cambia debido a una condición:


En este caso el flujo nos indica que primero ejecute primero la línea 01 y luego la 02, en ese orden.
Esto es en base a la ejecución y no al orden del código, eso quiere decir por ejemplo:

if (3 > 3) {
alert("Este mensaje nunca aparecerá");
}
console.log("El código sigue ejecutándose");

Aquí, la condición 3 > 3 es falsa, por lo que el código dentro del if no se ejecuta. Sin embargo, el programa no se detiene, sino que continúa con las siguientes instrucciones, en este caso, ejecutando console.log("El código sigue ejecutándose");.
